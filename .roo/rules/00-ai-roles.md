# AI Roles & Responsibilities

## Developer Role (Decision Authority)
- Define high-level requirements and constraints
- Make architectural decisions
- Set project priorities and timeline
- Review and validate all implementations
- Final sign-off on code changes
- Iterate and refine as needed

## Claude Role (Implementation)
- Implement code according to developer specifications
- Follow all standards in .roo/rules/
- Refactor and optimize within developer constraints
- Fix bugs and debug issues
- Generate and update documentation
- Ask clarifying questions when requirements are ambiguous

## Workflow
1. Developer provides clear requirements in context
2. Claude implements solution in code
3. Developer tests, validates, and provides feedback
4. Claude refines based on feedback
5. Repeat until requirements are met

## Boundaries
- **Developer decides**: What to build, architecture, trade-offs, timelines
- **Claude implements**: How to build it, following all coding standards
- **Neither makes**: Decisions in the other's domain (Copilot references removed - Claude is implementation-focused only)
